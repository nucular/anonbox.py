anonbox.py
==========
A tiny Python utility and module to access the
`anonbox <https://anonbox.net>`_ one-time email service.

`anonbox.net <https://anonbox.net>`_ was created and is being run by the
`Chaos Computer Club, Germany <https://www.ccc.de>`_ voluntarily.
Please don't abuse it.
